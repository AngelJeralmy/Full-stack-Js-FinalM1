Aquí podrás conocer un poco de mis trabajos hasta este momento. Puedes observar una breve descripción y, junto a esta, una imagen de cada una de mis creaciones en este entorno.

Descripción Muestra

Introducción a HTML: en este primer desafío del modulo uno, estudiamos la estructura de un documento con la extensión ".html", sus caracteristicas, sus partes y diferente componentes. Finalmente, se nos solicitó crear un pequeño Curriculum Vitae. Esta fue mi creación.

Introducción a CSS: en este segundo desafío del modulo uno, analizamos la composición, herramientas y funcionalidades de este lenguaje. Así pues, se nos solicito la creación de una pequeña pagina web usando las herramientas de CSS vistas en clase hasta el momento.

Flexbox: para el tercer desafío de este modulo, analizamos un poco en profundidad esta herramienta de posicionamiento. Es importante destacar, esta permite una distribución de los contenedores hijos de una manera mucho más expedita, lo que reduce el tiempo de creación exponencialmente. Tambien, se abordaron tópicos como la incorporación de íconos.

Bootstrap: cuarto desafío.

Se abordó un analisis medio para este "framework" de CSS. Analizamos clases como "d-flex" y sus diferente comandos para lograr un posicionamiento en contenedores de manera más sencilla. Tambien, se estudiaron clases para el fondo de color, color de texto, tamaño de letras, espacio horizontal y vertical entre las columnas de una rejilla, entre otras. Finalmente, se nos solicitó crear una "Landing page", responsiva, con base en una maqueta proporcionada.
Desktop Movil

Git, Gthub y terminal: desafío cinco.

Finalmente, el último desafío de este modulo consistió en hacer "Fork" de un repositorio, hacia un repositorio en nuestra cuenta Github. Luego, mediante el estudio e implementación de funciones de Git y la terminal, debiamos clonar este proyecto hacia nuestro equipo para finalizar haciendo cinco "commits" del proyecto "forkeado". Puedes observar el antes y despues.
